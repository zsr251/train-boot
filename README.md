## 项目说明

- 采用SpringBoot3.3、SpringSecurity6.3、Mybatis-Plus开发的培训班管理系统。
- 采用前后端分离的方式，前端使用Vue3、TypeScript、Element Plus、Vue Router、Pinia、Axios、i18n、Vite
- [视频演示](https://www.bilibili.com/video/BV1YgkMY4Ebg/)
